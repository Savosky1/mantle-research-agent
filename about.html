The Builder

Built with Purpose, Not Just Code

Onchain research is broken. Finding accurate, structured insights about RWAs, tokenized assets, and ecosystem movements takes hours of digging across X, dashboards, and whitepapers. The Mantle Research Agent fixes that. One input. One structured brief. Everything you need to think clearly about what's happening onchain.

How It Works

01

You Enter a Topic

Type any RWA asset, onchain trend, or Mantle ecosystem topic into the agent input field.

02

The Agent Analyzes

The AI processes your query against a knowledge base built around Mantle's ecosystem, RWA data, AI Agent Skills stack, and live onchain context pulled in real time.

03

You Get a Brief

A clean, structured research brief returns summary, market context, Mantle data points, key insights, and what comes next.

What Powers This

Frontend

HTML, CSS, and vanilla JavaScript. No frameworks. Fast, clean, and fully controlled. Every line written from scratch.

AI Brain

Groq AI (LLaMA 3.3 70B) via the Groq API, with a system prompt engineered specifically around Mantle's ecosystem, RWA data, AI Agent Skills, x402 payments, ERC-8004 identity, and onchain finance research. Live market data is injected into every brief in real time.

Live Onchain Data

Real-time MNT price from CoinGecko and Mantle TVL from DefiLlama, pulled directly and refreshed every 5 minutes. No static numbers, no guessing.

Secure Backend

API calls run through a Netlify serverless function. The API key never touches the browser. Visitors get the full experience without ever seeing what's behind it.

Hosting

Deployed on Netlify with continuous deployment from GitHub. Every push goes live automatically.

Access Control

Time-based session limits using localStorage. 20 research queries per 24 hours, resetting automatically so every visitor gets a fresh experience.

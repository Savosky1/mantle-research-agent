The Builder

Built with Purpose, Not Just Code

Onchain research is broken. Finding accurate, structured insights about RWAs, tokenized assets, and ecosystem movements takes hours of digging across X, dashboards, and whitepapers. The Mantle Research Agent fixes that. One input. One structured brief. Everything you need to think clearly about what's happening onchain.

How It Works

01

You Enter a Topic

Type any RWA asset, onchain trend, or Mantle ecosystem topic into the agent input field.

02

The Agent Analyzes

The AI processes your query against a knowledge base built around Mantle's ecosystem, RWA data, AI Agent Skills stack, and live onchain context pulled in real time.

03

You Get a Brief

A clean, structured research brief returns summary, market context, Mantle data points, key insights, and what comes next.

What Powers This

Frontend

HTML, CSS, and vanilla JavaScript. No frameworks. Fast, clean, and fully controlled. Responsive across desktop and mobile.

AI Brain

Claude AI via the Anthropic API. The agent's system prompt is engineered specifically around Mantle's ecosystem, RWA data, and onchain finance research including Mantle's full AI Agent Skills stack.

Live Onchain Data

Real-time MNT price and Mantle TVL pulled directly from CoinGecko and DefiLlama. Data refreshes every 5 minutes automatically.

Secure Backend

API calls run through a Netlify serverless function. The API key never touches the browser. Built the right way from the start.

Hosting

Deployed on Netlify with continuous deployment from GitHub. Every update ships automatically the moment code is pushed.

Access Control

Time-based session limits using localStorage. 5 research queries per 24 hours. The builder stays in control of the product even after submission.
